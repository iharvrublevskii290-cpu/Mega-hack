## Installation

1. Download the latest release
2. Install [Geode](https://geode-sdk.org/) if you haven't already
3. Open Geometry Dash and go to the Mods menu
4. Click "Install Mod" and select the downloaded file
5. Restart Geometry Dash

## Usage

1. Launch Geometry Dash with Geode
2. Open the mod menu (default key: `F1`)
3. Navigate through the tabs to access different cheat categories
4. Click on any cheat to toggle it on/off
5. Use the Settings tab to save/load your configuration

## Controls

- **F1** - Open/Close Cheat Menu
- **F2** - Toggle God Mode
- **F3** - Toggle Infinite Orbs
- **F4** - Toggle Super Speed
- **F5** - Save Settings
- **F6** - Load Settings

## Requirements

- Geometry Dash 2.2+
- Geode 2.0+
- Geode SDK 2.0+
